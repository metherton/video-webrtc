Twilio RTC SDKs - JavaScript Conversations Quickstart

Overview

Using the Twilio RTC SDKs, developers can add real-time communications to a wide variety of applications across multiple platforms. By harnessing the Twilio Conversations service, the Twilio RTC JavaScript SDK enables you to:

  • Invite another Web or Mobile SDK user to join you in an audio/video Conversation
  • Receive Conversation Invites from other Twilio RTC SDK users
  • Invite an additional Endpoint to an existing Conversation, creating a multi-party Conversation

In this tutorial, we build a basic implementation of a multi-party video communications app in the browser.

Prerequisites

  • A browser that supports WebRTC. During the JavaScript RTC SDK beta, official support is provided for the latest stable release of Mozilla Firefox and Google Chrome.

  • A Twilio Account and access to the Twilio account portal.
